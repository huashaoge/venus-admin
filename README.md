# 后台管理
### 简介
参考open-cloud 后端微服务的基础角色权限设计<a href="https://github.com/liuyadu/open-cloud">传送门</a>
此项目为单体项目，作为简单的前后端分离脚手架的后端项目，主要提供oauth2的简单认证授权功能，以及基础角色权限功能。
### 使用
默认账户: admin 123456

### 开发运行
依赖redis存储token信息，请先安装redis,开发与传统springboot开发没有区别，初始化sql后，修改配置运行即可
